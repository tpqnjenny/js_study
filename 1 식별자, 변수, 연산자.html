식별자

식별자 = 값을 저장, 이후 재활용 (문자, 배열, 함수, 객체 등)
    숫자 시작 불가
    특수문자 $ _ 사용 가능
    예약어 (for foreach var const let break continue 등)

변수

    var 변수명 = 값; 선언하면서 값을 초기화(할당), 어디에서든 사용 가능, 함수 내에서 지역변수, 호이스팅(끌어올려짐)
    let 변수명 = 값; 블록범위 지역변수 {}
    -- 값을 지정 후 변경 가능
    const 변수명 = 값; const 상수, 변경 불가

변수 타입

    기본타입: 숫자, 문자, 불리언(참거짓), null, undefined, 원시데이터, 단순화 할 수 없는 값
    참조타입: 복잡적인 값, 배열, 함수, 정규표현식, 객체 등

연산자

산술 연산자

    +, -, *, / 더하기, 빼기, 곱하기, 나누기
    + 문자 이후에 문ㄴ자 또는 숫자는 이어서 붙여준다
    % 나누기 연산 후 나머지 값
    ++ 1씩 증가
    -- 1씩 감소
    +=, -+, /=, %= 연산과 동시에 값을 할당
